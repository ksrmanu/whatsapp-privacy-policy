Privacy Policy

This application is used by NS Hospital to communicate with registered patients via WhatsApp. It is used to send lab reports, appointment reminders, hospital event updates, and other healthcare-related messages.

Information We Collect

  • Patient name and mobile number
  • Medical record number (MRNO)
  • Message delivery and status logs

How We Use This Information

The information is used solely for providing updates to patients. We do not share or sell this data to third parties.

Data Security

All communication is done via the official WhatsApp Business API with appropriate security standards.

Consent

By registering your number with NS Hospital, you consent to receive messages through WhatsApp.

Contact Us

If you have any questions, please contact us at: support@nshospital.org
